# DeepSeek Prompt — Fuelio CSV Import Backend (M3.2)

## Context

Majordom is a personal finance assistant. The SQLite schema already has `vehicles` and `vehicle_log` tables in `backend/core/memory/database.py`. Now implement the Fuelio CSV import backend.

**Critical rule:** Fuelio import = historical data → `vehicle_log` ONLY. **Zero AB transactions.** The user's Actual Budget only tracks finances from today forward; historical vehicle costs do not affect the budget.

**Architecture:** SQLite `vehicle_log` holds all operational history (odometer, liters, costs, reminders). AB holds only current/future financial transactions.

## 1. MemoryDB — vehicle methods (add to `database.py`)

### `upsert_vehicle(data: dict) -> int`

Upsert by `(name, plate)`. If exists → update fields, return existing id. If not → insert, return new id.

Fields from `data`: `name`, `make`, `model`, `year`, `plate`, `tank_capacity`, `fuel_type`, `active` (default 1).

```python
def upsert_vehicle(self, data: dict) -> int:
    conn = self._get_conn()
    try:
        row = conn.execute(
            "SELECT id FROM vehicles WHERE lower(name)=lower(?) AND lower(plate)=lower(?)",
            (data.get("name",""), data.get("plate",""))
        ).fetchone()
        if row:
            vid = row["id"]
            conn.execute("""
                UPDATE vehicles SET make=?,model=?,year=?,tank_capacity=?,fuel_type=?,active=?
                WHERE id=?
            """, (data.get("make"), data.get("model"), data.get("year"),
                  data.get("tank_capacity"), data.get("fuel_type"), data.get("active",1), vid))
            conn.commit()
            return vid
        cursor = conn.execute("""
            INSERT INTO vehicles (name,make,model,year,plate,tank_capacity,fuel_type,active)
            VALUES (?,?,?,?,?,?,?,?)
        """, (data.get("name"), data.get("make"), data.get("model"), data.get("year"),
              data.get("plate"), data.get("tank_capacity"), data.get("fuel_type"), data.get("active",1)))
        conn.commit()
        return cursor.lastrowid
    finally:
        conn.close()
```

### `insert_vehicle_log_entries(entries: list[dict]) -> tuple[int, int]`

Batch INSERT OR IGNORE using the UNIQUE(vehicle_id, fuelio_unique_id, entry_type) constraint already on the table. Returns `(inserted, skipped)`.

```python
def insert_vehicle_log_entries(self, entries: list[dict]) -> tuple[int, int]:
    conn = self._get_conn()
    try:
        inserted = 0
        for e in entries:
            cursor = conn.execute("""
                INSERT OR IGNORE INTO vehicle_log
                  (vehicle_id, date, odo_km, entry_type, fuel_liters, fuel_price_per_liter,
                   fuel_full_tank, fuel_missed, cost_total, cost_currency, remind_odo,
                   remind_date, repeat_odo, repeat_months, location, notes,
                   source, fuelio_unique_id)
                VALUES (?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?,?)
            """, (
                e.get("vehicle_id"), e.get("date"), e.get("odo_km"), e.get("entry_type"),
                e.get("fuel_liters"), e.get("fuel_price_per_liter"), e.get("fuel_full_tank"),
                e.get("fuel_missed"), e.get("cost_total"), e.get("cost_currency","EUR"),
                e.get("remind_odo"), e.get("remind_date"), e.get("repeat_odo"),
                e.get("repeat_months"), e.get("location"), e.get("notes"),
                e.get("source","fuelio_import"), e.get("fuelio_unique_id"),
            ))
            inserted += cursor.rowcount
        conn.commit()
        skipped = len(entries) - inserted
        return inserted, skipped
    finally:
        conn.close()
```

### `get_vehicles() -> list[dict]`

```python
def get_vehicles(self) -> list[dict]:
    conn = self._get_conn()
    try:
        rows = conn.execute("SELECT * FROM vehicles ORDER BY active DESC, name").fetchall()
        return [dict(r) for r in rows]
    finally:
        conn.close()
```

## 2. `backend/api/fuelio_import.py` — new file

### Endpoint

```
POST /api/import/fuelio
Content-Type: multipart/form-data
field: file (UploadFile)
```

Auth: `Depends(get_current_user)`.

### Response model

```python
class FuelioImportResult(BaseModel):
    vehicle_name: str
    fuel_entries: int
    fuel_skipped: int
    cost_entries: int
    cost_skipped: int
```

No AB proposals — historical data goes to vehicle_log only.

### Parser logic

Fuelio CSV has sections separated by `"## SectionName"` lines. Iterate line by line:

1. When line (stripped of quotes) starts with `## ` → set `current_section`
2. Known sections: `Vehicle`, `Log`, `CostCategories`, `Costs`
3. Ignore all others (`FavStations`, `Pictures`, `TripLog`, `Routes`, `Category`, etc.)
4. First data line after section header = CSV header for that section
5. Use Python `csv.reader` to parse individual lines, not the whole file

**Detection:** valid Fuelio file has `"## Vehicle"` as first line (stripped of quotes and whitespace).

### CostTypeID → entry_type mapping

```python
COST_TYPE_MAP = {
    "1": "service", "2": "maintenance", "4": "other",
    "5": "other", "6": "other", "7": "other",
    "8": "other", "9": "maintenance", "31": "insurance",
}
```

Default for unknown IDs: `"other"`.

### Vehicle section

Column mapping:
- `Name` → `name`
- `Make` → `make`, `Model` → `model`, `Year` → `year` (int)
- `Plate` → `plate`
- `Tank1Capacity` → `tank_capacity` (float)
- `fuel_type` = `"petrol"` always (FuelType 100/110/111 are all petrol variants)
- `active` = 1 (user manages active/inactive status manually after import)

Call `db.upsert_vehicle(data)` → get `vehicle_id`.

### Log section

Column mapping (note: Romanian header `Data` not `Date`):
- `Data` → `date` (already "YYYY-MM-DD HH:MM" format — store as-is)
- `Odo (km)` → `odo_km` (float, None if empty)
- `Fuel (litres)` → `fuel_liters` (float)
- `Full` → `fuel_full_tank` (int: "1"→1, "0"→0)
- `Price (optional)` → `cost_total` (float or None — **this is total paid, not price/liter**)
- `VolumePrice` → `fuel_price_per_liter` (float or None)
- `City (optional)` → `location`
- `Notes (optional)` → `notes`
- `Missed` → `fuel_missed` (int)
- `UniqueId` → `fuelio_unique_id` (store as string)
- `entry_type = "fuel"`, `source = "fuelio_import"`

### Costs section

Column mapping:
- `CostTitle` → `notes`
- `Date` → `date`
- `Odo` → `odo_km` (float; 0 → None — no odometer recorded)
- `CostTypeID` → `entry_type` via `COST_TYPE_MAP`
- `Cost` → `cost_total` (float — may be 0.0 for reminder-only entries)
- `RemindOdo` → `remind_odo` (0 → None)
- `RemindDate` → `remind_date` (see helper below)
- `RepeatOdo` → `repeat_odo` (0 → None)
- `RepeatMonths` → `repeat_months` (0 → None)
- `UniqueId` → `fuelio_unique_id`
- `source = "fuelio_import"`

```python
def _parse_remind_date(val: str) -> str | None:
    """'2011-01-01' is Fuelio's placeholder for 'no reminder' → None."""
    if not val or val.strip() == "2011-01-01":
        return None
    return val.strip()[:10]  # keep YYYY-MM-DD only
```

### Endpoint skeleton

```python
@router.post("/import/fuelio", response_model=FuelioImportResult)
async def import_fuelio(
    file: UploadFile = File(...),
    current_user: str = Depends(get_current_user),
):
    raw = await file.read()
    if len(raw) > 2 * 1024 * 1024:
        raise HTTPException(400, "File too large (max 2MB)")

    text = raw.decode("utf-8-sig", errors="replace")  # handle BOM
    lines = text.splitlines()

    if not lines or lines[0].strip().strip('"') != "## Vehicle":
        raise HTTPException(400, "Not a Fuelio sync file")

    # parse sections, upsert vehicle, batch-insert log + cost entries
    # return FuelioImportResult with counts only
```

## Gotchas

1. **`Data` not `Date`** in Log section header (Fuelio Romanian locale).
2. **UniqueId** is integer in file, store as string.
3. **`Price (optional)`** = total paid (not per-liter). May be empty string → None.
4. **`RemindDate "2011-01-01"`** = Fuelio placeholder → None.
5. **`Odo=0` in Costs** = no odometer → store None.
6. **`Cost=0.0`** = reminder-only entry (ITP, RCA) → save in vehicle_log, no AB transaction.
7. Ignore sections: `FavStations`, `Pictures`, `Category`, `TripLog`, `Routes`.
8. UNIQUE constraint `(vehicle_id, fuelio_unique_id, entry_type)` ensures Log and Costs don't collide even when both have UniqueId=1 (entry_type differs: "fuel" vs "service" etc.).
9. `csv.reader` on individual lines — NOT on whole file — because sections have different columns.
