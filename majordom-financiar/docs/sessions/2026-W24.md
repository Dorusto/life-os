### Lessons
- DeepSeek prompts: spec + gotchas only, no full implementation. Violation repeated 2026-06-12 — rule reinforced in memory.
- Tool disambiguation requires both: (1) explicit rule in system prompt AND (2) description that mentions what NOT to do. Missing system prompt rule = LLM falls back to closest-sounding tool.
- Local LLM viability at scale requires Option B (hierarchical routing). Option A (single LLM, prefixed tools) works for cloud. Prefixes make Option B a non-destructive add-on later.
